# Support

Choose the appropriate support option:

- API integration questions: [SocQ documentation](https://docs.socq.ai)
- Example bugs and requests: [`SocQAPI/socq-examples` issues](https://github.com/SocQAPI/socq-examples/issues)
- Account, billing, or private support: [support@socq.ai](mailto:support@socq.ai)
- API catalog and availability: [socq.ai/apis](https://socq.ai/apis)

Do not include API keys, customer data, production task payloads, or private
account information in public issues.
