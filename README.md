# SocQAPI organization defaults

This repository contains the public SocQ organization profile and the default
community health files inherited by repositories in the organization.

- The organization homepage is maintained in [`profile/README.md`](profile/README.md).
- Security reports must follow [`SECURITY.md`](SECURITY.md).
- General support requests should follow [`SUPPORT.md`](SUPPORT.md).
- Contributions should follow [`CONTRIBUTING.md`](CONTRIBUTING.md).

This repository never stores API keys, customer data, production task payloads,
private service configuration, or internal SocQ source code.
